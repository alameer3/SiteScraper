 قم بتطوير اداة الاستخراج الي كل هذه الميزات ابدا بالفحص لاتقوم باي شئ الا بعد موافقتي 
1. استخراج الواجهة الكاملة
• جميع ملفات HTML و CSS و JavaScript
• الصور والأيقونات والخطوط والملفات الصوتية/المرئية
• ملفات التصميم (SVG، PNG، JPG، WebP)
• ملفات التكوين (JSON، XML، Config files)
2. استخراج البنية التقنية
• قواعد البيانات وهيكلها (إن أمكن الوصول إليها)
• APIs المستخدمة والـ endpoints
• دوال JavaScript والمنطق البرمجي
• نظام التوجيه (Routing system)
• المكونات التفاعلية (Forms، Modals، Dropdowns)
3. استخراج الوظائف والميزات
• آلية المصادقة والتسجيل
• نظام إدارة المحتوى
• وظائف البحث والتصفية
• التنقل والقوائم
• الرسوم البيانية والتفاعل
• نظام التعليقات أو التقييمات (إن وجد)
4. استخراج سلوك الموقع
• JavaScript Events والتفاعلات
• AJAX calls وتحديث المحتوى
• Local Storage واستخدام Cookies
• Responsive behavior وتصميم الهاتف
• Loading states وإدارة الأخطاء
خطة التطوير المقترحة:
المرحلة الأولى: محرك الاستخراج العميق
• Spider Engine: زاحف ذكي يتنقل عبر كامل الموقع
• Asset Downloader: تحميل جميع الملفات والموارد
• Code Analyzer: تحليل وفهم الكود والمنطق البرمجي
• Database Scanner: محاولة فهم هيكل قاعدة البيانات
المرحلة الثانية: مولد الموقع المطابق
• Template Generator: إنشاء قوالب مطابقة للتصميم
• Function Replicator: إعادة إنشاء الوظائف والتفاعلات
• Asset Organizer: تنظيم الملفات في هيكل مشروع
• Code Generator: إنشاء كود مطابق للوظائف
المرحلة الثالثة: نظام الذكاء الاصطناعي
• AI Code Understanding: فهم منطق البرمجة بالذكاء الاصطناعي
• Pattern Recognition: تحديد أنماط التصميم والوظائف
• Smart Replication: إعادة الإنشاء الذكية للميزات المعقدة
• Quality Assurance: التحقق من دقة النسخ المُنشأة
الأدوات التقنية المتقدمة المطلوبة:
1. محركات استخراج متعددة:
• Playwright: لاستخراج المواقع التفاعلية والـ SPAs
• Selenium: للمواقع المعقدة والتفاعل العميق
• Trafilatura: لاستخراج النصوص والمحتوى
• BeautifulSoup: للتحليل التفصيلي للـ HTML
2. أدوات تحليل الكود:
• AST Parser: لتحليل JavaScript وفهم المنطق
• CSS Parser: لاستخراج وفهم ملفات التصميم
• Network Analyzer: لمراقبة طلبات الشبكة والـ APIs
3. مولدات الكود:
• Template Engines: لإنشاء قوالب مطابقة
• Code Synthesizers: لإعادة إنشاء الوظائف
• Asset Processors: لمعالجة وتحسين الملفات
